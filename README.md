### **Getting started:**

### _Prerequisites -_

#### _The software needed to install -_

- _Dependencies:_

- React Typescript

  ```sh
  npx create-react-app taskIt --template typescript
  ```

- React Beautiful DND

  ```sh
  npm i @types/react-beautiful-dnd
  ```

---

### **Installation:**

1. Clone the repo 

   ```sh
   git clone https://github.com/HMoana/Task-It.git
   ```

2. Install NPM packages

   ```sh
   npm install
   ```
